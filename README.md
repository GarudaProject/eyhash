# eyhash

<div align="center">
<p>Simple hashing module and CLI program for easy file and directory hashing.</p>
</div>

## Hashing

- MD5
- SHA1
- SHA256
- SHA512

## Usage

```sh
$ eyhash <filename/directory> -f/-d [file/directory]

# Output

```
